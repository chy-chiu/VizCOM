Fast Fourier Transform

The FFT Window calls numpy.fft.fft on the signal at each pixel.

  • FFT Parameters

    The Start Time & End Time parameters define the width of the FFT window

    The Subintervals parameter defines the number of subintervals to calculate. By default, each subinterval is evenly spaced over the interval defined by Start Time & End Time. Subintervals can be adjusted by clicking and dragging the vertical lines in the preview window AFTER selecting the desired number of subintervals.

  • FFT Output

    Peak Freqency Tab

    The Min & Max parameters define the minimum and maximum values of the colormap for the Peak Frequency Tab.

    The Interval # parameter sets the subinterval that will be displayed in the Peak Frequency Tab

    FFT Tab

    The FFT Tab displays the FFT output at the selected pixel. The peak frequency value is marked.
