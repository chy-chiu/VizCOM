Spatial Averaging

The Time Average function uses SciPy.ndimage to apply a 2-D convolution kernel with dimensions (2 * radius + 1)2 to each image.

  • Uniform mode

    Uniform mode applies averaging via a uniform kernel. sigma is ignored.

    -- Convolve with Uniform -- >

    See uniform_filter for more details.

  • Gaussian mode

    Gaussian mode applies averaging via a gaussian kernel, and a standard deviation of √(sigma/2).

    -- Convolve with Gaussian -- >

    See gaussian_filter for more details.
