Baseline Drift Removal

The Baseline Drift Removal function attempts to flatten each pixel's signal. First, it finds relative minima in each signal and displays the detected baseline points from confirmation. After confirmation, the detected baseline is subtracted from the corresponding signal.

  • Baseline Point Finding

    To find baseline points, we search each pixel's signal for relative minima using SciPy.signal's find_peaks function.

    The prominence parameter defines the minimum prominence a point must have in order to be considered for the baseline. Prominence of a given point is equal to the depth of the valley relative to the lowest neighboring peak

    The threshold parameter defines the maximum y value that will be considered for the baseline.

    The distance parameter defines the minimum time required between baseline points. If two baseline points are found within distance ms of each other, the lower of the two points will be kept.

  • Baseline Interpolation & Removal

    Once the user confirms the detected baseline points, baselines are constructed via linear interpolation between baseline points. The first and last baseline points are extended horizontally to beginning and end of the signal, respectively.

    These baselines are then subtracted from the signal at each pixel. Values that lie below the baseline are set to 0.
