Time Averaging

The Time Average function uses SciPy.ndimage to apply a 1-D convolution kernel with a width of 2 * radius + 1 to the signal at each pixel coordinate.

  • Uniform mode

    Uniform mode applies averaging via a uniform kernel. sigma is ignored.

    -- Convolve with Uniform -- >

    See uniform_filter for more details.

  • Gaussian mode

    Gaussian mode applies averaging via a gaussian kernel, and a standard deviation of sigma.

    -- Convolve with Gaussian -- >

    See gaussian_filter for more details.
